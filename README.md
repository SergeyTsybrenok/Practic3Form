# Сайт по формам:

https://sergeytsybrenok.github.io/Practic3Form/

## Запуск локально

> [!NOTE]
> C этим проектом рекомендуется использовать pnpm

### Зависимости
Install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

### Запуск сервера
Start the development server

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
